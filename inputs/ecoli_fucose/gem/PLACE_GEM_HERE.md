# Escherichia coli BL21(DE3) — GEM XML

Place the SBML Level 3 FBC v2 genome-scale metabolic model here.

## Expected file

- **Format:** SBML Level 3 FBC v2 (`.xml`)
- **Organism:** *Escherichia coli* BL21(DE3)
- **Recommended:** iML1515 or iECBD_1354 (`.xml`)

## How to obtain

Download iML1515 from BiGG Models: http://bigg.ucsd.edu/models/iML1515

## Usage

The pipeline uses this GEM for:
- L-fucose biosynthetic pathway grounding
- Stoichiometric equation table generation
- Hypothesis validation against E. coli metabolic network
